# Portfolio

Ce projet est un site web portfolio personnel pour présenter mes compétences et mes projets.

## Contenu

-   `index.html`: La page d'accueil.
-   `apropos.html`: La page "À propos".
-   `competences.html`: La page "Compétences".
-   `projets.html`: La page "Projets".
-   `contact.html`: La page "Contact".
-   `style.css`: La feuille de style principale.
-   `script.js`: Le script JavaScript principal.
-   `assets/`: Ce dossier contient les images, les fichiers CSS et JS spécifiques à chaque page, ainsi que le CV.
